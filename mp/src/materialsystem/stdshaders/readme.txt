Install Strawberry Perl and add it to your system path variable.

Open CMD and type:
perl -MCPAN -e shell

Then in the cpan shell type:
install String::CRC32

Edit the paths in buildmomentumshaders.bat to fit your setup.

After all that, you should be able to run buildmomentumshaders.bat successfully
Building PBR can take like half an hour, good luck.
